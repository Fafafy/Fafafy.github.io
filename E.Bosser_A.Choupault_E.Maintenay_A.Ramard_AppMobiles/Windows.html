Windows

Les applications sous Windows Phone sont développées en C# (prononcé C Sharp).

Le C# est dérivé du C++ et est très proche du Java dont il reprend la syntaxe générale. Il a été développé par Microsoft afin de le destiner à développer sur la plateforme Microsoft .NET. C# n’est pas bien différent de Java. En réalité sa création est surtout due à un désaccord entre Sun Microsystems et Microsoft qui entraîna un procès et une rupture entre les deux entreprises.

Exemple de code : "Hello Word"

        // A Hello World! program in C#.
        
        using System;
        namespace HelloWorld

            class Hello

On remarque que cela ressemble beaucoup à Java et peu au langage C.
