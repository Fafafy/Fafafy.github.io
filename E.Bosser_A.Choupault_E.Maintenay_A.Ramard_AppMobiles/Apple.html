Apple

iOS est la plateforme mobile d’Apple, les applications sous iOS sont développées en Swift mais aussi en Objective-C. Swift est un langage de programmation compilé multi-paradigmes développé par la société Apple. Il est destiné à la programmation d'applications sur les systèmes d'exploitation iOS, macOS, watchOS et tvOS. Il est conçu pour coexister avec l'Objective-C, langage actuel de programmation pour les systèmes d'exploitation d'Apple et le C. Le Swift est relativement jeune puisque sa première version publique est sortie en 2014. Ce langage est jugé « plus simple que l'Objective-C, permettant d'en faire autant en moins de lignes. »

Exemple de code en Swift : "Hello Word" 

        print("Hello, world!")

On remarque que le code est tout simple et qu’il ne nécessite aucune classe, aucune fonction déclarée au préalable. Objective-C est un langage de programmation orienté objet. C'est une extension du C, comme le C++, mais qui se distingue de ce dernier par sa distribution dynamique des messages. Aujourd’hui, il est surtout utilisé pour le développement sur les os Apple.

Exemple de code en Objective-C : "Hello Word" 

        int main (int argc, const char * argv[])
        {
            NSAutoreleasePool *pool = [[NSAutoreleasePool alloc] init];
            NSLog (@"Hello, World!");
            [pool drain];
            
            return 0;
        }

On remarque beaucoup de similarités au langage C, même si la synthaxe n'est pas la même.
