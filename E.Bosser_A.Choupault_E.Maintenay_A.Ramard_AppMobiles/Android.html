Android

Les applications pour Android sont développées en Java. Le java est un langage de programmation informatique orienté objet1 créé par James Gosling et Patrick Naughton, employés de Sun Microsystems2, avec le soutien de Bill Joy (cofondateur de Sun Microsystems), présenté officiellement en 1995.

Exemple de code : "Hello Word"

        public class HelloWorld
        {
            public static void main(String[] args)
            {
                System.out.println("Hello world!");
            }
        }

Ce qui permet à du code d’être exécuté de façon indépendante de la plateforme, c’est l’utilisation d’une machine virtuelle1. Justement, Java en utilise une, mais Google n’utilise pas celle de Java… Google (propriétaire d'Android depuis 2005) a créé sa propre machine virtuelle pour Android : la Dalvik Virtual Machine2. En temps normal, lorsque l’on code en Java, le code source est transformé en bytecodeJava3. Sous Android, le code source est compilé en bytecode. Il est propre à Google. Ce bytecode est exécuté par la Dalvik Virtual Machine, une machine virtuelle optimisée pour les plateformes avec une faible puissance. Ainsi, on aurait très bien pu coder en C par exemple, du moment que ce langage aurait été compilé sous la forme du bytecode propre à celui de Google.
